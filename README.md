# CodeAlpha Network Sniffer

This is my Task-1 submission for CodeAlpha Cyber Security Internship.

## Features
Captures live packets  
Shows Source / Destination IP  
Detects protocol (TCP, UDP, ICMP)  
Shows packet size  
Built with Python & Scapy  

## How To Run

### Windows
  pip install scapy
  python sniffer.py   
### Linux
  sudo apt install python3-pip
  pip install scapy
  sudo python3 sniffer.py
